# Product specification

ReplayCS addresses the gap between recognizing an answer and predicting execution. Its audience is undergraduate CS learners and self-taught developers. Its promise: **train students to predict what the computer will do next.**

The loop is: inspect state → predict → reveal deterministic execution → animate the mutation → explain → record mastery. Subjects are DSA I, DSA II, DBMS, Operating Systems, and Computer Networks. The MVP ships binary-search and compound-SQL traces, C/C++/Java/Python semantic mappings, predictions, XP, local progress, and an optional grounded AI mentor.

The demo opens binary search, predicts `mid`, advances and reverses exact states, changes language without restarting, then traces JOIN → WHERE → GROUP BY → aggregate → sort → limit. Gamification rewards first-time prediction and lesson completion; it never replaces feedback.

Out of scope: arbitrary code/SQL execution, accounts, leaderboards, all curriculum topics, production databases, complete OS/network modules, and AI-generated execution truth.

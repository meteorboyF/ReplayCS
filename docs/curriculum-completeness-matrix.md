marked expansion-complete yet: Binary Search now has case-matched evidence in the foundation lab,

## Status interpretation

- **Expansion contract partial/incomplete:** a real public lesson or foundation interface exists,
  but one or more required operation variants, complexity cases, exact work/space evidence, source
  mappings, prediction/recovery behaviors, or automated paths are missing.
- **Hidden planned metadata:** the topic remains in `subjects.ts` for roadmap tracking but is absent
  from `LIVE_LESSONS` and primary subject navigation. Registry and browser honesty tests protect
  that boundary; they are not tests of an unimplemented lesson.
- **Expansion-complete:** reserved for a topic that satisfies the complete universal lesson contract
  with deterministic tests. No current row qualifies.

Static complexity labels in an existing lesson do not count as a complexity explanation. Family
reference cards likewise do not promote their example algorithms into completed curriculum topics.
